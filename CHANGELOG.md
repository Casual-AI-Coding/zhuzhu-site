# Changelog

## [1.9.0] - 2026-03-21

### ✨ 新功能

#### 愿望清单
- 新增愿望清单功能，支持与 Notion 数据库同步
- 日历视图：按月展示愿望，区分进行中/已完成颜色
- 筛选器：支持状态（进行中/已完成）和分类筛选
- 完成操作：添加完成日期选择弹窗
- 卡片展示：目标日期、分类标签、状态标识
- 悬浮操作按钮（FAB）快速添加愿望
- 空状态引导设计

#### Notion 集成增强
- 支持 Notion Status 类型字段
- 适配 rich_text 类型标题字段
- 新增愿望清单数据库配置项 `VITE_NOTION_DATABASE_WISHES`

### 🔧 改进

- `useDaysCount` 重构为引用计数单例管理共享定时器
- 移动端筛选器改为图标显示，节省空间
- 统一筛选器为悬浮面板设计
- 完成按钮采用玻璃拟态风格
- 优化 Vue 组件生命周期和 DOM 操作

### 🐛 修复

- GuestbookView 修复重复 onMounted 注册问题
- GalleryView 重置触摸状态防止照片切换时状态泄漏
- 修复刷新按钮不调用 Notion API 的问题
- 修复 FAB 按钮与回到顶部按钮重叠
- 移除所有 debug console 语句
