# 愿望清单功能设计文档

**日期**: 2026-03-19  
**功能**: 愿望清单（一起完成的愿望）  
**状态**: 已设计，待实现

---

## 1. 功能概述

两人共同维护的愿望清单，记录"一起想做的事"，当愿望成真时记录完成日期，像纪念日一样珍藏。

---

## 2. Notion 数据模型

### 数据库: Wishes (愿望清单)

| 字段 | 类型 | 说明 |
|------|------|------|
| 标题 | title | 愿望名称，如"一起去日本" |
| 描述 | rich_text | 详细描述/备注，可选 |
| 分类 | select | 旅行 / 美食 / 体验 / 家居 / 其他 |
| 目标日期 | date | 年月（可选） |
| 优先级 | select | 高 / 中 / 低 |
| 状态 | select | 进行中 / 已完成 |
| 完成日期 | date | 实际完成的日期 |

## 4. 进行中视图

### 4.1 筛选栏

- **分类**: 全部 / 旅行 / 美食 / 体验 / 家居 / 其他
- **优先级**: 全部 / 高 / 中 / 低
- **重置筛选** 按钮
- 移动端默认折叠，点击"筛选"展开

### 4.2 卡片列表

- 按优先级排序（高 > 中 > 低）
- 每张卡片显示:
  - 左侧优先级色条（高=红色🔴，中=橙色🟠，低=绿色🟢）
  - 标题（大字）
  - 分类标签 🏷️
  - 目标日期（如果有）📅
  - 描述（截断两行，hover 时展开）
- 底部"✨ 成真了！"按钮

### 4.3 动画特效

- **入场动画**: 卡片 staggered fade-in + slide-up（每张延迟 50ms）
- **标记成真**: 卡片缩小 + 飘出金光粒子效果 → 消失
- **空状态**: 漂浮的✨动画 + 鼓励文案"写下你们的第一个愿望吧~"

## 5. 已完成视图

### 5.1 视图切换器

📋 列表视图 | 📅 日历视图

两个视图共享同一套筛选器（分类、优先级）

### 5.2 列表视图

- 瀑布流/网格卡片布局
- 每张卡片显示: 标题、分类、完成日期、描述
- 入场动画（同进行中）
- 悬停/点击: 轻微上浮 + 阴影加深

### 5.3 年日历视图

**年份切换器**: ◀ 2026 ▶

**月份格子布局**: 3×4（桌面端）/ 2×6（移动端）

每个格子显示:
- 月份标签（1月、2月...）
- 完成愿望数量徽章
- 当月有愿望完成时: 格子发光边框 + 背景微光

**动画**:
- 页面入场: 月份格子 staggered 缩放入场
- 有愿望的格子: hover 时微微放大 + 光晕脉冲
- 切换年份: 月份格子 flip 翻转过渡
- 愿望完成时: 格子闪光特效

点击月份格子: 弹出 Modal 显示该月所有完成的愿望

## 6. 添加/编辑愿望 Modal

### 6.1 触发

- 右上角 "+" 按钮
- 点击卡片进入详情/编辑

### 6.2 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标题 | text | ✓ | 最大 50 字符 |
| 描述 | textarea | - | 最大 500 字符 |
| 分类 | select | ✓ | 旅行/美食/体验/家居/其他 |
| 目标日期 | month-picker | - | 年-月选择器 |
| 优先级 | radio | - | 高/中/低，默认"中" |

### 6.3 操作按钮

- **保存**（主按钮）
- **取消**（次按钮）
- **删除**（仅编辑时，危险操作红色）

### 6.4 动画

- Modal 入场: 背景渐暗 + 内容 slide-up + fade
- 字段获焦: label 上浮 + 边框高亮
- 保存成功: Modal 缩放消失 + 卡片区域闪新✨

---

## 7. 愿望成真流程

1. 点击卡片底部"✨ 成真了！"按钮
2. 弹出完成日期选择器（默认当天）
3. 确认后:
   - 卡片播放"成真"动画（缩小 + 金光粒子飘散）
   - 愿望状态变为"已完成"，完成日期记录
   - 卡片从进行中列表移除
   - 若目标日期所在月份有日历视图，该月格子闪光
